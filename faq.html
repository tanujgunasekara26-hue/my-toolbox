❓ FAQ

Frequently asked questions about My Toolbox

💡 Click any question below to expand the answer. Can't find what you're looking for? Check the Troubleshooting page or report an issue.
📋 General Questions

What is My Toolbox?

▼
My Toolbox is a free, open-source desktop application for Windows that bundles 21 professional tools into one place. It includes AI chatbot, PDF tools, image editor, unit converter, password generator, timer, and much more — all in a beautiful, modern interface.

How much does My Toolbox cost?

▼
My Toolbox is completely free. No subscription, no ads, no paywalls. You can download it and use it forever without paying anything.

Is My Toolbox open-source?

▼
Yes! My Toolbox is open-source under the MIT License. You can view the source code, audit it for security, and contribute improvements on GitHub.

Does My Toolbox track or monitor me?

▼
No. My Toolbox has zero tracking, zero telemetry, and zero analytics. We have no way to know what you do or who you are. Your data stays on your computer.

Does My Toolbox require an account?

▼
No account needed. Install, open, and start using all 21 tools immediately. The only exception is if you want to use AI features — you'll need a free Cerebras API key.
💾 Installation & Setup

What are the system requirements?

▼
Minimum: Windows 10 or later, Intel Core i5 (2nd Gen) or better, 8GB RAM, 500MB free disk space.

Recommended: Windows 11, Intel Core i7 or better, 16GB RAM, 1GB free disk space.

Can I run My Toolbox on macOS or Linux?

▼
Currently, My Toolbox is Windows-only. macOS and Linux support are planned for future versions. You can run from source on any platform, but the compiled installer is Windows-only.

How do I install My Toolbox?

▼
Download the installer from the Download page, run it, follow the prompts, and you're done. Takes about 2 minutes. See the Installation Guide for detailed instructions.

Can I run My Toolbox from a USB drive?

▼
Not yet (planned for future versions). Currently, you need to install it on your computer. You can copy the installation folder to a USB drive, but it may not work without proper installation.
🛠️ Tools & Features

What are all 21 tools?

▼
See the Tools page for a complete list with descriptions. Categories include AI & Writing, Conversion, Utilities, Productivity, Media, Business, and System tools.

Can I use tools offline?

▼
Most tools work offline (Calculator, Timer, Stopwatch, Notes, Password Generator, Image Tools, etc.). Only AI Chatbot, AI Writing Tools, and Currency Converter require internet.

Can I add my own tools?

▼
The app doesn't support plugins yet, but you can contribute new tools to the GitHub repository. See Contributing for guidelines.
🤖 AI Features

How do I set up AI features?

▼
You need a free Cerebras API key. Sign up at cloud.cerebras.ai, generate a key, and set it as the CEREBRAS_API_KEY environment variable. See Setting Up Cerebras AI for step-by-step instructions.

Are AI features free?

▼
Yes! Cerebras offers a generous free tier. Most casual users won't exceed it. You only pay if you make thousands of API calls per month.

Where do I get a Cerebras API key?

▼
Go to cloud.cerebras.ai, create a free account, navigate to API Keys, and generate one. It takes 2 minutes. See Setting Up Cerebras AI for details.

Can I use AI features outside the US?

▼
No. AI features are restricted to users in the United States due to export regulations. There's no workaround — this is enforced at the API level. All other 19 tools work fine anywhere.

Is my data safe with Cerebras?

▼
Cerebras logs interactions for service improvement. Your prompts are never used for training without consent. See Cerebras's Privacy Policy and our Privacy Policy for details.
🔒 Data & Privacy

Where does My Toolbox store my data?

▼
All data is stored locally on your computer in %APPDATA%\MyToolbox\data\. This folder contains your notes, settings, and preferences. See Data & Storage for full details.

Can I back up my notes?

▼
Yes. Copy %APPDATA%\MyToolbox\data\notes.txt to any safe location (USB drive, cloud storage, external drive). This is a plaintext file you can open with any text editor.

Is my data encrypted?

▼
Not currently — data is stored as plaintext, protected only by Windows file permissions. Encryption is planned for future versions. For now, use Windows password protection on your user account.

Can I move the data folder?

▼
Not yet. The data folder is always in %APPDATA%\MyToolbox. This feature is planned for future versions.
🎨 Themes & Customization

How do I change the theme?

▼
Click Settings (⚙️) → Appearance → Theme → Select a theme → Apply. Changes apply instantly. See Themes for details on all 5 themes.

How many themes are there?

▼
5 themes: Light, Dark, Blue, Green, and Purple. Each has been professionally designed and tested for readability and aesthetics.

Can I create custom themes?

▼
Not yet through the UI, but since My Toolbox is open-source, you can modify the theme files and recompile. Check Contributing for guidelines.
📦 Updates & Support

How do I update My Toolbox?

▼
Download the latest installer from the Download page or GitHub Releases and run it over your existing installation. Your notes and settings are preserved.

How often are updates released?

▼
Updates are released when features are ready and bugs are fixed. There's no fixed schedule. Enable auto-check in Settings to be notified of new versions.

I found a bug. How do I report it?

▼
Go to the Report Issues page, describe the problem, and submit via GitHub. Include your OS version, My Toolbox version, and steps to reproduce.

I have a feature request. What should I do?

▼
Open a GitHub issue with the title "Feature Request:" and describe your idea. Check existing issues first to avoid duplicates. See GitHub.
