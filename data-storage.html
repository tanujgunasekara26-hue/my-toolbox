💾 Data & Storage

How My Toolbox handles your data — locally, privately, and securely

📋 Privacy-First Philosophy

Your data stays on your computer. My Toolbox is designed with privacy as the foundation:

  • ✅ No telemetry or analytics
  • ✅ No tracking or profiling
  • ✅ No account or login required
  • ✅ No data collection or sharing
  • ✅ All data stored locally

The only exceptions are when you explicitly use features that require internet (AI, currency conversion).

🗂️ Where Data is Stored

My Toolbox stores all local data in a single folder on your computer.

Default Location

%APPDATA%\MyToolbox\data\

This translates to:

C:\Users\[YourUsername]\AppData\Roaming\MyToolbox\data\

How to Find It

  1. Press Win + R
  2. Type: %APPDATA%\MyToolbox
  3. Press Enter
💡 The AppData folder is hidden by default. If you can't see it, enable "Show hidden files" in File Explorer View settings.

📝 What Gets Stored?

Here's what My Toolbox stores locally:

Data Type Stored Where Format Purpose
User Notes data/notes.txt Plaintext Notes you write in the Notes tool
Settings data/settings.json JSON Theme, font size, preferences
App Preferences data/config.ini INI Window size, recent tools, etc.
Cache data/cache/ Various Temporary files for performance
Tool Data data/tools/ Various Data saved by individual tools
✅ All stored data is in plain text or standard formats you can inspect yourself.

🔒 Security Considerations

Local Storage is Safe

  • File Permissions: Data is stored in your user folder, only accessible to your user account
  • No Encryption: Data is plaintext (future versions may add encryption)
  • No Network Access: Your data never leaves your computer (except for AI/currency features)

Protect Your Data

  • 🔐 Use Windows password protection on your account
  • 🔐 Enable Windows Defender or another antivirus
  • 🔐 Don't paste sensitive data (passwords, API keys) into My Toolbox
  • 🔐 Back up important notes regularly
💡 If you're concerned about privacy on a shared computer, consider creating a separate Windows user account.

🗑️ Clearing Your Data

Clear Cache

To clear temporary files without losing your notes or settings:

  1. Open My Toolbox Settings (⚙️)
  2. Go to Advanced
  3. Click "Clear Cache"

Reset to Defaults

To reset all settings to factory defaults:

  1. Settings → Advanced
  2. Click "Reset to Defaults"
  3. This keeps your notes but resets preferences

Delete Everything

To remove all My Toolbox data:

  1. Uninstall My Toolbox via Control Panel (Programs and Features)
  2. Manually delete the folder: %APPDATA%\MyToolbox
⚠️ This is permanent. Deleted notes and data cannot be recovered. Back up important data first.

☁️ What About Cloud Sync?

Currently: My Toolbox does NOT sync to the cloud. All data stays local.

Future: Cloud sync may be added as an optional, privacy-respecting feature (encrypted, client-side). This will always be opt-in.

Manual Backup

To back up your notes:

  1. Open the data folder: %APPDATA%\MyToolbox\data\
  2. Copy notes.txt to a safe location (USB drive, cloud storage like OneDrive, etc.)

🌐 Internet-Connected Features

Some features send data to external services. Here's what happens:

🤖 AI Chatbot & AI Writing Tools

  • Destination: Cerebras Cloud servers
  • What's Sent: Your prompt/text only
  • Returned: AI-generated response
  • Logging: Cerebras logs interactions for service improvement
  • Privacy: See Cerebras Privacy Policy

💱 Currency Converter

  • Destination: Frankfurter API (no-auth service)
  • What's Sent: Currency codes you're converting
  • Returned: Exchange rates
  • Logging: Minimal (standard API logs)
  • Privacy: No personal data collected
💡 See our Privacy Policy for details on third-party API data handling.

📊 File Size & Storage

My Toolbox data is small. Here are typical storage sizes:

  • notes.txt: 5-50 KB (depends on how many notes)
  • settings.json: 2-5 KB
  • config.ini: 1-2 KB
  • Cache: 1-10 MB (cleared automatically)
  • Total: Typically under 20 MB

You'll never run out of disk space with My Toolbox.

❓ FAQs

Q: Does My Toolbox track me?

A: No. Zero tracking, zero analytics, zero telemetry. We have no way to know what you do.

Q: Can you read my notes?

A: No. Notes are stored locally on your computer, never sent anywhere.

Q: Is my data encrypted?

A: Currently, no. Data is stored as plaintext in your local user folder (protected by Windows permissions). Encryption may be added in future versions.

Q: What happens when I uninstall?

A: The installer will ask if you want to keep your data. You can delete it or keep it for re-installation later.

Q: Can I move the data folder?

A: Not yet (planned for future versions). Currently, it's always in the standard AppData location.

Q: What if I lose my data?

A: Back up your notes regularly. Copy notes.txt to a safe location or cloud service.
