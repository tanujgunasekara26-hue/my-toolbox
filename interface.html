🖥️ The Interface

Navigate, customize, and master My Toolbox's interface

Dashboard Overview

When you launch My Toolbox, you'll see the main dashboard — your central hub for all 21 tools.

Dashboard Elements

Tool Cards

Grid of all 21 tools, each with an icon and quick description. Click to open.

Top Menu Bar

Home, Settings, Help, and About buttons for quick access.

Search Bar

Quickly find tools by typing their name.

Status Bar

Shows current theme, version, and system status at the bottom.

Navigation

Moving around My Toolbox is straightforward.

From the Dashboard

  • Open a Tool: Click any tool card to launch it immediately
  • Go Back: Click the Home button or press Escape (tool-dependent)
  • Search Tools: Type a tool name in the search bar to filter the grid
  • Settings: Click the Settings icon (gear icon) to customize appearance and behavior

Inside a Tool

  • Back to Dashboard: Click the Home button or use the back arrow (usually top-left)
  • Close Tool: Click the X button in the top-right corner
  • Keyboard Shortcuts: Many tools support keyboard shortcuts — check the Help menu inside each tool
💡 Pro Tip: Right-click on a tool card from the dashboard for quick actions like "Pin to Taskbar" or "Run on Startup" (if available).

Settings & Customization

Make My Toolbox feel like yours with themes, language, and behavior settings.

Accessing Settings

  1. Click the Settings icon (⚙️) from the dashboard or top menu
  2. Browse categories on the left sidebar (Appearance, Behavior, Advanced, About)
  3. Adjust preferences and apply immediately

Available Settings

Appearance

  • Theme: Choose from 5 built-in themes (Light, Dark, Blue, Green, Purple)
  • Font Size: Increase or decrease text size globally (100%, 110%, 120%)
  • Show Grid Lines: Toggle decorative grid background on/off
  • Animations: Enable smooth transitions between screens

Behavior

  • Always on Top: Keep My Toolbox above other windows
  • Remember Window Position: Save and restore window size/location on startup
  • Confirm Before Close: Ask before closing the app (prevents accidents)
  • Auto-Check for Updates: Notify when a new version is available

Advanced

  • Clear Cache: Remove temporary files and reset certain tools
  • Data Storage Path: View or change where notes and preferences are saved
  • Reset to Defaults: Restore all settings to factory defaults
  • Enable Developer Mode: Show debug logs and performance metrics (for troubleshooting)
💡 Tip: Settings are saved automatically. You don't need to click "Save" after making changes.

5 Built-In Themes

Status Bar & System Tray

The status bar at the bottom of My Toolbox shows useful information at a glance.

Status Bar Information

  • Current Theme: Shows which theme is active
  • Application Version: Your installed version number (e.g., v2.1)
  • System Status: CPU & RAM usage, time until auto-lock, etc.
  • Connection Status: Indicates if internet is available (relevant for AI & currency tools)

System Tray Icon

My Toolbox adds an icon to your system tray (bottom-right taskbar) for quick access:

  • Single-click: Show or hide the window
  • Right-click: Quick menu (Settings, About, Exit)
💡 To disable the system tray icon, go to Settings → Behavior and toggle Show in System Tray off.

Keyboard Shortcuts (Global)

These shortcuts work from any screen in My Toolbox:

  • Ctrl + H — Go to Home / Dashboard
  • Ctrl + S — Open Settings
  • Ctrl + F — Focus the search bar
  • Ctrl + Q — Quit My Toolbox
  • Escape — Go back (context-dependent)
  • F1 — Open Help
  • F11 — Toggle fullscreen (if supported)
💡 Individual tools may have their own shortcuts — check the Help menu inside each tool.

Tips & Tricks

🎯 Stay Organized

  • Use the search bar to quickly find tools instead of scrolling
  • Pin frequently-used tools to the taskbar (right-click tool card)
  • Group related tools in your mind (all AI tools, all converters, etc.)

🎨 Personalize Your Experience

  • Switch themes based on time of day (Light during day, Dark at night)
  • Adjust font size for comfort (Settings → Appearance → Font Size)
  • Enable "Always on Top" if you frequently reference data while working

⚡ Work Efficiently

  • Use keyboard shortcuts to navigate faster
  • Keep My Toolbox minimized in the system tray and toggle it quickly
  • Enable auto-check for updates so you never miss bug fixes
