🔧 Troubleshooting

Common issues and solutions

Installation Issues

❌ Windows SmartScreen Warning

Issue: Windows blocks the installer saying "Windows protected your PC"

Cause: Fresh builds sometimes trigger false positives. Totally harmless.

Solution:

  1. Click "More info" on the warning
  2. Click "Run anyway" at the bottom right
  3. Installation proceeds normally

❌ Installer Won't Start / "File Corrupted"

Cause: Download was incomplete or corrupted

Solution:

  1. Delete the downloaded .exe file
  2. Re-download from the Download page
  3. Verify file size matches (should be ~50-80 MB)
  4. Try installing again

❌ "Permission Denied" During Install

Cause: Insufficient permissions or file in use

Solution:

  1. Right-click installer → "Run as administrator"
  2. Close all other running instances of My Toolbox
  3. Try again

Launch & Crash Issues

❌ App Won't Start / Crashes Immediately

Cause: Missing dependencies or corrupted installation

Solution:

  1. Uninstall My Toolbox via Control Panel → Programs → Programs and Features
  2. Delete folder: %APPDATA%\MyToolbox
  3. Restart your computer
  4. Re-download and install the latest version

❌ "Module Not Found" Error

Cause: Python dependency missing (if running from source)

Solution:

pip install -r requirements.txt --break-system-packages

Then restart My Toolbox.

❌ App Hangs / Freezes on Startup

Cause: Corrupted settings or slow system

Solution:

  1. Force close My Toolbox (Ctrl+Alt+Delete, Task Manager)
  2. Delete: %APPDATA%\MyToolbox\data\config.ini
  3. Re-open My Toolbox (it will rebuild settings)

Performance Issues

🐌 App is Slow / Laggy

Cause: Low system resources or background processes

Solution:

  • Check CPU/RAM in System Info tool (built into My Toolbox)
  • Close other heavy applications (Chrome, video editors, etc.)
  • Restart your computer
  • Check for Windows updates and install them

🐌 QR Scanner is Slow

Cause: Your webcam or CPU is slow, or poor lighting

Solution:

  • Ensure good lighting on the QR code
  • Hold the code steady for 2-3 seconds
  • Try scanning from a file instead of webcam
  • Close other apps using the webcam

AI & Internet Issues

🤖 AI Tools Show "Authentication Failed"

Cause: Cerebras API key not set or incorrect

Solution:

  1. Verify you set the CEREBRAS_API_KEY environment variable
  2. Confirm the key is correct (copy from Cerebras Cloud again if unsure)
  3. Restart My Toolbox after setting the environment variable
  4. See Setting Up Cerebras AI for detailed steps

🤖 AI Tools Show "Region Restricted"

Cause: You're outside the United States

Solution: AI features are U.S.-only due to export regulations. No workaround is permitted. Use the other 19 tools instead.

💱 Currency Converter Shows "No Data"

Cause: No internet connection or API is down

Solution:

  1. Check your internet connection
  2. Try again in a few seconds
  3. If persists, check Frankfurter API status

⏳ AI Responses are Slow

Cause: Network latency or API load

Solution:

  • Check your internet speed (should be at least 5 Mbps)
  • Cerebras API may be under heavy load — wait a moment and try again
  • Shorter prompts usually respond faster

Tool-Specific Issues

📷 QR Scanner Says "No Camera Detected"

Solution:

  1. Check Device Manager (Win + X → Device Manager)
  2. Look for your webcam under "Cameras"
  3. If marked with ⚠️, update drivers
  4. Make sure My Toolbox has camera permissions (Settings → Privacy → Camera)
  5. Restart My Toolbox

🖼️ Image Tools Won't Open

Cause: Missing image file or unsupported format

Solution:

  • Only supports: PNG, JPG, BMP, GIF
  • File must exist and not be in use by another app
  • Try selecting a different image first

📄 PDF Tools Say "File Corrupted"

Cause: PDF is truly corrupted or not a valid PDF

Solution:

  • Try opening the PDF in Adobe Reader first to verify it's valid
  • If Reader can't open it, the PDF is corrupted — recreate it
  • Ensure you have file permissions (not read-only)

📝 Notes Won't Save

Cause: File permissions issue or disk full

Solution:

  • Check available disk space: C: drive should have at least 100 MB free
  • Ensure folder permissions: %APPDATA%\MyToolbox should be writable
  • Try copying your notes to Notepad, then back into My Toolbox

Display & Theme Issues

🎨 Theme Won't Change

Solution:

  1. Go to Settings → Appearance
  2. Select a different theme
  3. Click "Apply" (or it applies automatically)
  4. Restart My Toolbox if it doesn't update

📺 UI Text is Blurry or Too Small

Solution:

  • Go to Settings → Appearance → Font Size
  • Increase to 110% or 120%
  • Restart My Toolbox

🖥️ App Opens Off-Screen

Solution:

  1. Right-click My Toolbox in taskbar
  2. Select "Move"
  3. Press arrow keys to reposition
  4. Or delete: %APPDATA%\MyToolbox\data\config.ini to reset position

Data & Settings Issues

💾 Notes Disappeared!

Cause: Accidental deletion or cleared cache

What to Try:

  1. Check %APPDATA%\MyToolbox\data\notes.txt — if file is there but empty, recovery may be difficult
  2. Check Windows File History or Recycle Bin
  3. Unfortunately, if not backed up, recovery may not be possible

Prevention: Back up notes regularly to a safe location.

⚙️ Settings Keep Resetting

Cause: File permissions or corrupted config

Solution:

  1. Close My Toolbox
  2. Delete: %APPDATA%\MyToolbox\data\config.ini
  3. Reopen My Toolbox and reconfigure settings

  2. Describe the problem in detail
  3. Include: OS version, My Toolbox version, steps to reproduce
  4. Submit via GitHub Issues
