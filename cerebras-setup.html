🔧 Setting Up Cerebras AI

Complete guide to get AI features working in My Toolbox

Overview

The AI Chatbot and AI Writing Tools need a Cerebras API key to function. This guide walks you through getting a key and setting it up on your Windows system.

⚠️ Important: AI features are available in the United States only. Attempting to bypass this with a VPN violates the Terms of Service.

Step 1: Create a Cerebras Account

If you don't already have one, sign up for a free Cerebras account.

  1. Go to cloud.cerebras.ai
  2. Click "Sign Up" or "Get Started"
  3. Choose your sign-up method (email, Google, GitHub, etc.)
  4. Fill in account details (name, email, password)
  5. Verify your email if prompted
  6. Complete any additional setup steps
💡 Free Cerebras accounts include a monthly API quota. Check your limits at the dashboard.

Step 2: Generate an API Key

Once logged in, generate an API key that My Toolbox will use to communicate with Cerebras.

  1. Log in to cloud.cerebras.ai
  2. Navigate to API Keys (look for a menu item, sidebar, or account settings)
  3. Click "Create API Key" or "Generate New Key"
  4. Give it a name (e.g., my-toolbox) to remember what it's for
  5. Click Create or Generate
  6. Your API key will appear — copy it immediately (most services only show it once)
⚠️ IMPORTANT: Treat your API key like a password. Never share it publicly, commit it to GitHub, or paste it in chat logs. Keep it secret.

Step 3: Add the Key as an Environment Variable

My Toolbox reads your API key from an environment variable named CEREBRAS_API_KEY. Set this on Windows using one of the methods below.

Method A: Using the System Properties GUI (Easiest)

  1. Press Win (Windows key) on your keyboard
  2. Type "Environment Variables" to search
  3. Click "Edit the system environment variables"
  4. A window titled "System Properties" will open. Click "Environment Variables..." button (bottom-right)
  5. Under "User variables for [YourUsername]", click "New..."
  6. Set:
    Variable name: CEREBRAS_API_KEY Variable value: [paste your API key here]
  7. Click OK on all open windows
  8. Restart My Toolbox (or any terminal/IDE) so it picks up the new variable
✅ The environment variable is now set. You can verify it worked in the next step.

Method B: Using Command Prompt

If you prefer the command line:

  1. Press Win + R to open Run
  2. Type cmd and press Enter
  3. Run this command (replace with your actual key):
    setx CEREBRAS_API_KEY "your-api-key-here"
  4. You should see: SUCCESS: Specified value was saved.
  5. Close the command prompt window
  6. Restart My Toolbox to apply the change

Method C: Using PowerShell

For PowerShell users:

  1. Press Win + X and select "Windows PowerShell" (or PowerShell Core)
  2. Run this command:
    [System.Environment]::SetEnvironmentVariable("CEREBRAS_API_KEY", "your-api-key-here", "User")
  3. Close PowerShell
  4. Restart My Toolbox to apply the change

Step 4: Verify the Setup

Check that the environment variable was set correctly.

Option 1: Using Command Prompt

  1. Press Win + R and type cmd, then Enter
  2. Run:
    echo %CEREBRAS_API_KEY%
  3. If it prints your API key (or the first few characters), ✅ it's set correctly.
    If it prints %CEREBRAS_API_KEY% unchanged, ❌ the variable isn't set. Try again.

Option 2: Using Python

  1. Open Command Prompt or PowerShell
  2. Run:
    python -c "import os; print(os.environ.get('CEREBRAS_API_KEY'))"
  3. If it prints your key, ✅ it's set correctly.
    If it prints None, ❌ the variable isn't set. Retry the steps above.

Step 5: Confirm It Works in My Toolbox

Test the AI features to ensure everything is configured properly.

  1. Restart My Toolbox (close and reopen the app)
  2. Click the AI Chatbot tool card
  3. Type a test message: Hello! Can you hear me?
  4. Press Enter or click Send
  5. If Claude responds, ✅ you're done! The API key is working.
    If you get an error, see the Troubleshooting section below.
✅ Congratulations! Your Cerebras API key is set up and working. You now have full access to My Toolbox's AI features.

Step 6: Optional — Adding Cerebras to PATH (For Developers)

If you're running My Toolbox from source and want to use Cerebras tools directly from your terminal, add the Python Scripts folder to your PATH:

  1. First, install the SDK:
    pip install cerebras-cloud-sdk --break-system-packages
  2. Find your Python Scripts folder (typically):
    C:\Users\[YourUsername]\AppData\Local\Programs\Python\Python3XX\Scripts
  3. Open System Properties → Environment Variables (see Step 3 above)
  4. Under "User variables", select Path → click "Edit..."
  5. Click "New" and paste the Scripts folder path
  6. Click OK on all windows
  7. Open a new terminal window and verify:
    python -c "import cerebras"

Troubleshooting

❌ "Invalid API Key" Error

Cause: The key is wrong or expired.

Solution:

  • Go to cloud.cerebras.ai and generate a new API key
  • Update your CEREBRAS_API_KEY environment variable with the new key
  • Restart My Toolbox

❌ "Feature Not Available in Your Region"

Cause: You're not in the United States.

Solution: AI features only work for US-based users. If you're outside the US, you cannot use this feature.

❌ Environment Variable Not Working

Cause: The variable wasn't set correctly or the app wasn't restarted.

Solution:

  • Verify the variable is set: Run echo %CEREBRAS_API_KEY% in Command Prompt
  • If it's not there, repeat Step 3 (setting the environment variable)
  • Completely close and reopen My Toolbox — simply clicking Home isn't enough
  • If you were running from a terminal, close and reopen the terminal too

❌ Slow Responses or Timeouts

Cause: Internet issues or Cerebras servers under heavy load.

Solution:

  • Check your internet connection
  • Try a simpler prompt (longer requests take more time)
  • Wait a few minutes and try again
  • Check Cerebras status page for outages

❌ "Rate Limit Exceeded"

Cause: You've used up your monthly API quota.

Solution:

  • Check your usage at cloud.cerebras.ai
  • Free tier limits reset monthly
  • Consider upgrading your Cerebras plan if you need more quota

❌ Still Not Working?

Report the issue on GitHub with:

  • The exact error message
  • Your Windows version (check Settings → System → About)
  • Whether you're using the .exe or running from source
  • Steps to reproduce the issue
