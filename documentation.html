🚀 Getting Started

My Toolbox is a desktop application built with Python and PySide6, bundling 21 productivity, creativity, and system tools into a single, themeable interface.

💡 Tip: No account, sign-up, or internet connection is required for most tools. A handful of tools (AI Chatbot, AI Writing Tools, Currency Converter) need internet access to reach their respective APIs.

💾 Installation

Option 1 — Windows Installer (Recommended)

  1. Go to the Releases page
  2. Download the latest MyToolbox-Setup.exe
  3. Run the installer and follow the on-screen steps
  4. Launch My Toolbox from the Start Menu or Desktop shortcut

Option 2 — Run From Source

git clone https://github.com/tanujgunasekara26-hue/my-toolbox.git
cd my-toolbox
pip install -r requirements.txt
python main.py

Requires Python 3.10+ and the dependencies listed in requirements.txt (PySide6, psutil, opencv-python, numpy, requests).

🖥️ The Interface

The app opens to a Home dashboard listing every available tool. Use the sidebar to navigate between tools at any time — your current tool stays open until you switch.

  • Sidebar: Quick navigation to all 21 tools plus the About page
  • Theme Switcher: Found in settings — choose from 5 built-in themes
  • Title Bar: Shows the active tool name

🤖 AI Features

The AI Chatbot and AI Writing Tools are powered by the Cerebras API.

  • Availability: United States only, due to API export restrictions
  • Internet: Required for every request
  • Data: Prompts are sent directly to Cerebras — see the Privacy Policy
⚠️ Attempting to bypass the geographic restriction with a VPN or proxy violates the Terms of Service.

🔧 Setting Up Cerebras AI

The AI Chatbot and AI Writing Tools need a Cerebras API key to function. Follow the steps below to get a key and make it available to My Toolbox.

1. Create a Cerebras Account

  1. Go to cloud.cerebras.ai
  2. Sign up for a free account (or sign in if you already have one)
  3. Verify your email if prompted

2. Generate an API Key

  1. From the Cerebras Cloud dashboard, open the API Keys section
  2. Click Create API Key, give it a name (e.g. my-toolbox), and confirm
  3. Copy the generated key immediately — most providers only show it once
💡 Tip: Treat your API key like a password. Never share it publicly, commit it to a public GitHub repo, or paste it into chat logs.

3. Add the Key as an Environment Variable (Windows)

My Toolbox reads the key from an environment variable named CEREBRAS_API_KEY. You can set this permanently using the Windows GUI or the command line.

Option A — Using the System Properties GUI:

  1. Press Win and search for "Environment Variables"
  2. Click "Edit the system environment variables"
  3. In the System Properties window, click "Environment Variables..."
  4. Under "User variables", click New...
  5. Set Variable name to CEREBRAS_API_KEY
  6. Set Variable value to your copied API key
  7. Click OK on all open windows to save
  8. Restart My Toolbox (and your terminal, if running from source) so it picks up the new variable

Option B — Using Command Prompt:

setx CEREBRAS_API_KEY "your-api-key-here"

setx sets the variable permanently for your user account. Close and reopen any terminal or the app for the change to take effect.

Option C — Using PowerShell:

[System.Environment]::SetEnvironmentVariable("CEREBRAS_API_KEY", "your-api-key-here", "User")

4. Adding Cerebras to PATH (Running from Source / CLI tools)

If you're running My Toolbox from source and also want to use the Cerebras CLI or SDK tools directly from any terminal, make sure the Python Scripts folder (where pip installs console scripts) is on your PATH:

  1. Install the Cerebras SDK: pip install cerebras-cloud-sdk --break-system-packages (or without the flag in a virtual environment)
  2. Find your Python Scripts folder, typically:
    C:\Users\<YourUsername>\AppData\Local\Programs\Python\Python3XX\Scripts
  3. Open Environment Variables (same screen as above)
  4. Under "User variables", select Path → click Edit...
  5. Click New and paste the Scripts folder path
  6. Click OK to save, then open a new terminal window to apply the change

Verify it worked by running:

python -c "import os; print(os.environ.get('CEREBRAS_API_KEY'))"

If your key prints out (rather than None), the environment variable is correctly set and My Toolbox's AI Chatbot and AI Writing Tools will be able to use it.

5. Confirm It's Working

  1. Launch My Toolbox
  2. Open the AI Chatbot tool
  3. Send a test message

If you see a response instead of an authentication error, you're all set.

⚠️ Remember: Cerebras-powered features are restricted to users in the United States regardless of whether your API key is valid. See AI Features above and the Terms of Service.

🎨 Themes

My Toolbox ships with 5 built-in themes covering light and dark variants. Switch anytime from the settings menu — your choice is remembered between sessions.

💾 Data & Storage

All local data (notes, preferences, generated files) is stored on your own device — nothing is uploaded to any server controlled by the Developer.

  • Notes are saved in plaintext, with no encryption
  • QR codes are not saved automatically unless you export them
  • App preferences persist locally between launches

For full details, see the Privacy Policy.

🔧 Troubleshooting

App won't open / antivirus flags the .exe

Freshly built executables are sometimes flagged as a false positive by antivirus software. This is a known issue — see the Known Issues list.

AI Chatbot says "unavailable in your region"

This is expected outside the United States. It is a hard restriction of the underlying Cerebras API, not a bug.

Currency Converter not updating

Check your internet connection. Rates are fetched live from the Frankfurter API and update daily, not in real time.

❓ FAQ

Is My Toolbox free?

Yes, completely free and open-source.

Does it collect my data?

No. There's no telemetry, analytics, or tracking. See the Privacy Policy for full details.

Can I use it on Mac or Linux?

The packaged installer targets Windows 10/11. Running from source with Python may work on other platforms, though it isn't officially supported.
