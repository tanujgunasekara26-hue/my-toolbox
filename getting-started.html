🚀 Getting Started

Start using My Toolbox in 5 minutes

⚡ Quick Setup (5 Minutes)

Get My Toolbox installed and exploring tools in just a few steps.

  1. Download My Toolbox

    Visit the Download page and grab the latest MyToolbox-Setup.exe

  2. Run the Installer

    Double-click the downloaded file and follow the on-screen prompts

  3. Launch the App

    Click the Desktop shortcut or find My Toolbox in your Start Menu

  4. Pick a Theme (Optional)

    Click Settings ⚙️ and choose from 5 themes. Restart to apply

  5. Explore a Tool

    Click any tool card and start using it immediately

✅ You're ready! All 21 tools are now available.

3. Customize Settings

  • Click the Settings icon (⚙️) from the top menu
  • Explore Appearance to change themes, font size, animations
  • Check Behavior for options like "Always on Top"
  • Settings apply immediately — no restart needed

4. Enable AI Features (Optional)

To use the AI Chatbot and AI Writing Tools:

  • Get a free Cerebras API key at cloud.cerebras.ai
  • Add it as an environment variable (see Setting Up Cerebras AI)
  • Restart My Toolbox
  • Open the AI Chatbot and start chatting
ℹ️ Note: AI features are only available in the United States due to API restrictions.

❓ Common Questions

Do I need an account to use My Toolbox?

No. My Toolbox is completely free and works offline. No account or registration required (except for optional AI features).

Can I use it on Mac or Linux?

My Toolbox is built for Windows 10+. Mac and Linux support is not currently available, but you can run from source if you're adventurous.

Is my data secure?

Yes. All data stays on your machine. We don't collect, track, or send any personal information (except to Cerebras/Frankfurter APIs if you use those specific tools).

Can I use the app without internet?

Yes, most tools work offline. Only the AI Chatbot, AI Writing Tools, and Currency Converter require internet.

What if I run into issues?

Check the Installation Guide troubleshooting section, or report the issue on GitHub.
