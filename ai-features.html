🤖 AI Features

Powered by Cerebras API — Intelligent Tools for Smart Work

What Are AI Features?

My Toolbox includes two AI-powered tools built on the Cerebras API, giving you access to Claude AI directly in your desktop application. These tools help with writing, brainstorming, coding, analysis, and much more.

The Two AI Tools

🤖 AI Chatbot

Chat with Cerebras AI. Ask questions, get advice, brainstorm ideas, write code, analyze data, or just have a conversation.

✍️ AI Writing Tools

Smart writing assistance including grammar checking, tone adjustment, summarization, paraphrasing, and content generation.

AI Chatbot Capabilities

The AI Chatbot allows you to have free-form conversations with Claude AI. Here's what you can ask it to do:

Category Examples
Writing & Editing Write emails, essays, articles; edit for clarity; improve tone
Programming Debug code; explain algorithms; generate code snippets; optimize performance
Analysis & Research Summarize articles; analyze data; research topics; compare concepts
Learning & Education Explain complex topics; create study guides; generate quiz questions
Brainstorming Generate ideas; find solutions; explore different perspectives
Productivity Create templates; outline projects; organize information

AI Writing Tools Capabilities

The AI Writing Tools provide specialized assistance for text-based tasks:

Grammar & Style Checking

  • Detect grammatical errors and suggest corrections
  • Identify spelling and punctuation mistakes
  • Check tone and readability
  • Suggest style improvements

Text Transformation

  • Paraphrasing: Rewrite text while preserving meaning
  • Summarization: Create concise summaries of longer content
  • Expansion: Develop ideas and add more detail
  • Tone Adjustment: Make text more formal, casual, professional, friendly, etc.

Content Generation

  • Generate email templates
  • Create social media posts
  • Write product descriptions
  • Generate brainstorm ideas

Important Limitations & Requirements

🌍 Geographic Restriction

⚠️ AI features are available in the United States only, due to Cerebras API export restrictions. If you're outside the US, the AI tools won't work even with a valid API key.

🔑 API Key Required

To use AI features, you need a free Cerebras API key. See Setting Up Cerebras AI for setup instructions.

📡 Internet Connection Required

AI tools need a live internet connection to communicate with Cerebras servers. All other My Toolbox tools work offline.

🔒 Privacy & Data

  • Your prompts are sent to Cerebras servers in real-time
  • See the Privacy Policy for details on data handling
  • Cerebras does not store conversations — they're processed and discarded

⏱️ Rate Limits

Free Cerebras accounts may have rate limits on API calls. Check your account dashboard at cloud.cerebras.ai for current limits.

How to Use AI Features

Step 1: Get an API Key

  • Sign up at cloud.cerebras.ai
  • Generate an API key in the dashboard
  • Copy the key (shown only once)

Step 2: Configure My Toolbox

  • Add the key as an environment variable: CEREBRAS_API_KEY
  • Follow the detailed guide in Setting Up Cerebras AI
  • Restart My Toolbox to pick up the key

Step 3: Open an AI Tool

  • Go to the My Toolbox dashboard
  • Click AI Chatbot or AI Writing Tools
  • Start using the tool immediately

Step 4: Use It

Type your prompt, question, or text and let Claude AI help you. Results appear in seconds.

Tips & Best Practices

For the AI Chatbot

  • Be specific: More details = better responses. Instead of "help me code", say "help me write a Python function that converts hex colors to RGB"
  • Provide context: Share relevant info or constraints
  • Ask follow-ups: Refine or expand on answers with follow-up questions
  • Use for brainstorming: Ask Claude to generate multiple ideas or approaches

For AI Writing Tools

  • Paste complete text: The tool works best with full passages, not fragments
  • Specify tone: Tell it what tone you want (formal, casual, creative, etc.)
  • Review suggestions: AI isn't perfect — always review and edit as needed
  • Iterate: Use multiple passes to refine text (summarize → expand → tone adjust)

Troubleshooting AI Features

❌ "Invalid API Key" Error

Solution: Your API key might be incorrect or expired. Go to cloud.cerebras.ai, generate a new key, and update your CEREBRAS_API_KEY environment variable.

❌ "Not Available in Your Region"

Solution: AI features only work in the United States. If you're outside the US, the feature won't work regardless of API key validity.

❌ Slow Responses or Timeouts

Solution: Check your internet connection. Large requests may take longer. Cerebras servers might be under heavy load — try again in a few minutes.

❌ Rate Limit Exceeded

Solution: You've hit your API quota. Check your usage at cloud.cerebras.ai. Free tier limits reset monthly.

❌ Feature Still Won't Work

Report the issue on GitHub with your error message and system details.
